datastruct for rust
=====================

A Rust library for creating datastruct. 

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
datastruct = "0.1"
```

and this to your crate root:

```rust
extern crate datastruct;
```

How to use
```rust
use datastruct::RBTree;
let mut m = RBTree::new();
assert_eq!(m.len(), 0);
m.insert(1, 2);
assert_eq!(m.len(), 1);
m.insert(2, 4);
assert_eq!(m.len(), 2);
assert_eq!(*m.get(&1).unwrap(), 2);
assert_eq!(*m.get(&2).unwrap(), 4);
```
